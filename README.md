# LoggerDemo

**Showcase how to define and use a custom Logger formatter**

## Usage

```elixir
LoggerDemo.test
```

Output
```
[info] LoggerDemo.test 
        Demo
        Basic
14:23:06 [debug] LoggerDemo.test 
        Demo
        Basic
14:23:06 [error] LoggerDemo.test 
        Demo
        Basic
:ok   
```
